# Library Management System

## Project Objective
To demonstrate problem-solving and coding skills by designing a system that manages book recordsâ€”allowing adding, searching, updating, and deleting, using database storage.

## Features
- Add new books to the library database.
- Search books by title.
- View all books in the collection.
- Update book details.
- Delete book records.

## Technologies Used
- Python 3.x
- SQLite3 for database
- pandas library for CSV import
- Google Colab or local Python for running the program

## How to Run
1. Download/clone the repository.
2. Ensure books.csv and library.db are present in the project folder.
3. Run VITYarthi_proj.py (or open the .ipynb Colab notebook).
4. Follow the menu prompts to perform library operations.

*Dependencies:*  
Run pip install pandas if not already installed.

## Usage Example
When program starts, you'll see a menu like:
1.Add Book
2.View All Books
3.Search Book By Title
4.Update Book
5.Delete Book
6.Exit

Enter your choice and follow the prompts.
